# Sequelized Burger

The Sequleized Burger App is a web app that lets users create, update, and delete burgers that they have eaten.

![Demo of Sequelized Burger App](public/assets/images/demo.gif)

## Languages/Technologies

- Express.JS

- Handlebars.JS

- MySQL

- Sequelize
